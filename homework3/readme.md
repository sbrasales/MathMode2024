# Это третье домашнее задание
 Программа состоит из трех основных частей:
- **server.py**: это файл, содержащий код для веб-сервера. Он определяет маршруты с использованием Flask и реализует функции для обработки HTTP-запросов, в данном случае отправки и получения сообщений от сервера 
- **sender.py**:этот файл действует как клиент для отправки сообщений на сервер. Это интерактивный скрипт, позволяющий пользователю вводить сообщения и отправлять их на сервер. Запрашивает у пользователя ввод имени и сообщений. Отправляет сообщения через запрос POST.
- **receiver.py**: получает входящие сообщения от сервера и отображает их в терминале пользователя.

Чтобы запустить проект, необходимо ввести в командной строке: `server.py`, а затем, независимо от порядка, `sender.py` и `receiver.py`. Можно открыть другие вкладки в терминале, чтобы смоделировать других пользователей.

Кроме того, запустив `server.py`, мы можем войти на различные маршруты, например, `/status`, где будет отображаться количество подключенных пользователей и отправленных сообщений.
